# rusoblanco

Roberto Abdelkader Martínez Pérez's personal blog.

Built with [Hugo](https://gohugo.io) using the [rusoblanco theme](https://github.com/nilp0inter/rusoblanco-hugo-theme).

## Building

```bash
hugo
```

## Local development

```bash
hugo server -D
```

## Deployment

The site is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the main branch.

The site is available at: https://www.rusoblanco.com/
